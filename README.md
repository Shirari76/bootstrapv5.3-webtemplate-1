# قالب وب Bootstrap فارسی

یک قالب وب کامل و ریسپانسیو با استفاده از Bootstrap 5.3 و پشتیبانی کامل از زبان فارسی.

## ویژگی‌ها

- ✅ طراحی کاملاً ریسپانسیو
- ✅ پشتیبانی کامل از RTL (راست به چپ)
- ✅ استفاده از Bootstrap 5.3.8
- ✅ فونت فارسی Vazir
- ✅ آیکون‌های Bootstrap Icons
- ✅ کاروسل تصاویر
- ✅ کارت‌های قیمت‌گذاری
- ✅ فرم تماس
- ✅ فوتر کامل با لینک‌های شبکه‌های اجتماعی

## بخش‌های قالب

### 1. هدر و منوی ناوبری
- منوی ریسپانسیو با dropdown
- برند و لوگو
- لینک‌های اصلی سایت

### 2. بخش معرفی (Hero Section)
- متن معرفی با دکمه‌های CTA
- تصویر شاخص

### 3. بخش خدمات
- کارت‌های خدمات با آیکون
- طراحی grid ریسپانسیو

### 4. بخش قیمت‌گذاری
- سه پکیج مختلف
- کارت ویژه با رنگ متمایز
- دکمه‌های خرید

### 5. گالری تصاویر
- کاروسل Bootstrap
- کنترل‌های ناوبری

### 6. فرم تماس
- فیلدهای ایمیل و پیام
- تصویر جانبی

### 7. فوتر
- اطلاعات شرکت
- لینک‌های مفید
- آیکون‌های شبکه‌های اجتماعی

## فایل‌های پروژه

```
├── index.html          # فایل اصلی HTML
├── style.css           # استایل‌های سفارشی
├── fonts/              # فونت‌های فارسی
│   ├── Vazir.eot
│   ├── Vazir.ttf
│   ├── Vazir.woff
│   └── Vazir.woff2
└── images/             # تصاویر و آیکون‌ها
    ├── img_1.png
    ├── img_2.jpg
    ├── img_3.jpg
    ├── img_4.jpg
    ├── img_5.png
    └── img_6-11.svg
```

## نحوه استفاده

1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید
3. برای سفارشی‌سازی، فایل `style.css` را ویرایش کنید

## تکنولوژی‌های استفاده شده

- **HTML5** - ساختار صفحه
- **CSS3** - استایل‌دهی
- **Bootstrap 5.3.8** - فریمورک CSS
- **Bootstrap Icons** - آیکون‌ها
- **Vazir Font** - فونت فارسی

## سفارشی‌سازی

### تغییر رنگ‌ها
رنگ‌های اصلی در فایل `style.css` تعریف شده‌اند:
- رنگ اصلی: `#1b2a4e`
- رنگ آیکون‌ها: `#0d6efd`
- پس‌زمینه آیکون‌ها: `#ebeafb`

### تغییر فونت
فونت Vazir در فایل CSS تعریف شده. برای تغییر فونت، مسیر فایل‌های فونت را در `@font-face` ویرایش کنید.
